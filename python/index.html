Python简介

Python是一种解释型、高级编程语言，由Guido van Rossum于1991年创建。Python以其简洁的语法和丰富的库生态系统而闻名，是初学者学习编程的理想选择。

Python的主要特点：

  • 易于学习和使用
  • 可读性强
  • 跨平台
  • 丰富的标准库
  • 强大的第三方库支持

Python安装

在开始学习Python之前，我们需要先安装Python环境。Python的安装非常简单：

  1. 访问Python官网 www.python.org
  2. 下载最新版本的Python安装包
  3. 运行安装程序，记得勾选"Add Python to PATH"选项
  4. 完成安装后，打开命令行输入 python --version 验证安装

基础语法

Python的基础语法非常简单直观：

# 这是一个简单的Python程序
print("Hello, World!")

# 变量定义
name = "Python"
age = 30

# 条件语句
if age > 20:
    print(f"{name} 是一个成熟的编程语言")

# 循环
for i in range(3):
    print(f"第{i+1}次循环")

变量和数据类型

Python中的主要数据类型包括：

  • 数字（整数、浮点数）
  • 字符串
  • 列表
  • 元组
  • 字典
  • 集合
# 数字
x = 10    # 整数
y = 3.14  # 浮点数

# 字符串
name = "Python"

# 列表
fruits = ["苹果", "香蕉", "橙子"]

# 字典
person = {
    "name": "张三",
    "age": 25
}
