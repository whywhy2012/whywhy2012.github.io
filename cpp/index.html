C++简介

C++是一种通用编程语言，支持多种编程范式，包括过程式、面向对象和泛型编程。它是C语言的超集，在保持C语言高效性的同时，增加了面向对象编程的特性。

C++的主要特点：

  • 高性能和效率
  • 灵活的内存管理
  • 丰富的标准库
  • 跨平台支持
  • 广泛的应用领域

开发环境配置

要开始C++编程，你需要：

  1. 安装编译器（如GCC、Visual Studio、CLion等）
  2. 配置IDE或文本编辑器
  3. 设置环境变量

推荐的IDE：

  • Visual Studio（Windows平台最佳选择）
  • CLion（跨平台IDE，需付费）
  • Code::Blocks（免费开源）

基础语法

C++程序的基本结构：

// 第一个C++程序
#include 
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

// 变量声明和使用
int age = 25;
double height = 1.75;
string name = "张三";

// 条件语句
if (age >= 18) {
    cout << "成年人" << endl;
} else {
    cout << "未成年" << endl;
}

变量和数据类型

C++的基本数据类型：

  • 整型（int、short、long、long long）
  • 浮点型（float、double）
  • 字符型（char）
  • 布尔型（bool）
  • 空类型（void）
// 数据类型示例
int number = 42;
double pi = 3.14159;
char grade = 'A';
bool isValid = true;

// 常量定义
const int MAX_SIZE = 100;

// 数组
int numbers[5] = {1, 2, 3, 4, 5};

// 字符串
string message = "Hello C++";
